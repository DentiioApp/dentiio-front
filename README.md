## 📋 Requirements
- 🛠Make
- ⚛️ React >= 16.12
- NodeJS >= 13.10
- 🐳Docker

## 🎉 Building your app

### with docker
0. Launch the command  `make help` or `make` generate list of targets with descriptions
1. `make run` for build the app
2. `make start` for start the app

### without docker
1. `yarn install` or `yarn`
- For run in production mode  `yarn build'
2. `yarn start`
